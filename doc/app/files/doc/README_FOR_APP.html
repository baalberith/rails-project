Strona przeznaczona do nauki języka japońskiego.

Niezarejestrowani użytkownicy mogą przeglądać: 

        - dodane słowa,
        - znaki kanji,
        - listy słówek innych użytkowników.

Zarejestrowani użytkownicy mogą ponadto: 

        - dodawać do bazy słowa ich czytania i znaczenia,
        - edytować czytania i znaczenia dodanych słów,
        - tworzyć listy słówek i dodawać do nich słówka,
        - usuwać swoje listy słówek,
        - edytować swój profil.

Admin może również: 

        - usuwać słowa i ich znaczenia

Gotowa strona na: glowing-mist-92.heroku.com
